# SinceraTech Company Site

Official marketing website for Sincera Tech Solutions — technology services for businesses in Malawi.

## Features

- Home, Services, Packages, and Contact pages
- Firebase-backed contact form with optional EmailJS notifications
- SEO meta tags, Open Graph, and JSON-LD structured data
- Responsive layout with motion-enhanced UI

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and fill in Firebase and EmailJS values (optional for email).
3. Start the dev server:
   ```bash
   npm run dev
   ```

The app runs at `http://localhost:3000`.

## Build

```bash
npm run build
npm run preview
```
